基于当前游戏代码分析，确定的标准尺寸：

| 类型 | 尺寸 | 用途 | 示例 |

## 🎨 **美术资源获取策略**

### 📥 **16x16资源处理**
很多优秀的像素艺术是16x16，需要放大：
```
16x16 → 32x32 (2倍放大)
推荐工具: 
- Photoshop: 图像大小 → 像素插值: 邻近
- GIMP: 图像 → 缩放图像 → 插值: 无
- 在线工具: waifu2x (保持像素艺术风格)
```

### 📥 **32x32资源**
理想尺寸，直接使用：
```
直接导入游戏即可
特别推荐: Kenney.nl 的32x32资源包
```

### 📥 **64x64+高分辨率**
过大需要缩小：
```
建议避免，缩小会丢失细节
除非专门为高分辨率制作
```

## 🛠️ **实际操作建议**

### 🚀 **快速开始 (30分钟)**
1. **下载 Kenney Tiny Dungeon** 
   - 已经是16x16，需要2倍放大
   - 选择最适合的角色sprite

2. **放大到正确尺寸**
   ```
   玩家角色: 16x16 → 40x40 (2.5倍)
   敌人角色: 16x16 → 32x32 (2倍)
   ```

3. **重命名并导入**
   ```
   player.png → assets/textures/characters/
   enemy.png → assets/textures/enemies/
   ```

### 🎯 **精确尺寸转换**

**从16x16 Kenney资源：**
```bash
# 使用ImageMagick批量处理
# 玩家角色 (16→40)
magick input.png -scale 250% -crop 40x40+4+4 player.png

# 敌人角色 (16→32) 
magick input.png -scale 200% enemy.png

# 道具图标 (16→24)
magick input.png -scale 150% item.png
```

**在线工具：**
- Photopea.com (免费在线PS)
- Pixlr.com 
- 设置: 最近邻插值/像素完美缩放
